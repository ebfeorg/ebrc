# 什么是 ebaorc

ebaorc目前包括，`ebn-fe-components`, `ebn-fe-hooks`, `ebn-fe-utils`。分贝是业务组件库，hooks，以及常用工具方法

## 特性

全新的 dumi 2.0 主要具备以下特性：

- 🚀 **组件库**：高度集成 `antd`，`@ant-design/pro-components`,与业务系统保持一致
- 🔍 **hooks**：常用hooks
- 🎨 **utils**：常用工具方法
